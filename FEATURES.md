# 仓库管理系统 - 新功能说明

## 🎨 界面优化

### 仓库显示优化
- **仓库名称显示**: 在仓库管理页面,仓库名称现在显示为两行,第一行为主要名称,第二行为地址信息
- **仓库选择器优化**: 在商品管理页面的仓库选择器中,下拉选项显示为"仓库名称 (地址)"格式,提高辨识度
- **命名规范优化**: 仓库命名采用"城市+区域+类型"格式,如"北京朝阳主仓"、"上海浦东分仓"、"广州天河备用仓"
- **重复数据验证**: 前后端均添加了仓库名称重复验证,避免创建同名仓库
- **示例优化后**: "北京朝阳主仓 (北京市朝阳区)"、"上海浦东分仓 (上海市浦东新区)"

## ✅ 已完成的功能

### 1️⃣ 条码/二维码扫描功能

#### 功能介绍
- 为商品添加条码字段,支持商品条码管理
- 支持通过条码快速搜索商品
- 为商品生成二维码,方便扫码出入库

#### 使用方法
1. **添加商品条码**:
   - 在商品管理页面,新建或编辑商品时,可以输入商品条码(选填)

2. **条码搜索**:
   - 在商品列表页面,输入条码并点击"查询"按钮
   - 系统会快速定位到对应的商品

3. **生成二维码**:
   - 点击商品列表中的"二维码"按钮
   - 系统会生成该商品的二维码,可打印后使用

#### 技术实现
- 后端:Goods实体添加barcode字段,新增`/goods/barcode/{barcode}`接口
- 前端:使用qrcode库生成二维码

---

### 2️⃣ 库存预警系统

#### 功能介绍
- 自动检测低库存商品
- 实时预警通知
- 预警状态管理(未读/已读)
- 手动检查库存

#### 使用方法
1. **查看预警**:
   - 点击左侧菜单"库存预警"
   - 可查看未读预警和所有预警

2. **手动检查**:
   - 点击"检查库存"按钮
   - 系统会扫描所有商品库存,生成预警

3. **处理预警**:
   - 点击"标记已读"标记预警已处理
   - 点击"删除"移除不需要的预警

#### 预警类型
- **低库存预警**: 当前库存低于10时自动触发

#### 技术实现
- 新建Alert实体和AlertController
- 支持实时预警和历史记录查询

---

### 3️⃣ Excel批量操作

#### 功能介绍
- 批量导入商品数据
- 导出商品列表为Excel
- 提供导入模板下载

#### 使用方法
1. **下载模板**:
   - 点击"下载模板"按钮
   - 按照模板格式填写商品数据

2. **批量导入**:
   - 点击右下角"批量导入"按钮
   - 拖拽或选择Excel文件上传
   - 系统自动解析并导入数据

3. **导出Excel**:
   - 点击"导出Excel"按钮
   - 下载当前商品列表的Excel文件

#### 模板格式
| 列名 | 说明 | 必填 |
|------|------|------|
| 商品名称 | 商品名称 | 是 |
| 仓库名称 | 仓库名称(需已存在) | 是 |
| 库存数量 | 初始库存数量 | 是 |
| 商品条码 | 商品条码 | 否 |
| 备注 | 商品备注 | 否 |

#### 技术实现
- 后端:新增ExcelController处理导入导出
- 前端:使用xlsx库处理Excel文件

## 📝 注意事项

1. **条码功能**:
   - 条码字段为选填,不输入不影响正常使用
   - 建议为重要商品设置条码,便于管理

2. **库存预警**:
   - 默认阈值为10,可在AlertController中调整
   - 定期执行"检查库存"以获取最新预警

3. **Excel导入**:
   - 请严格按照模板格式填写数据
   - 仓库名称必须已在系统中存在
   - 单次建议不超过1000条记录

4. **性能优化**:
   - 大量数据导出建议使用分页查询
   - 导入操作建议在业务低峰期执行
